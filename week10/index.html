Fetch Api:

The fetch api is a great way to use external sources including JSON files and obtain the information. Fetch is used through URLs and gives us the capability to send requests to pass information to be used on a web page. Then that information passed is known as the response. Other properties such as headers can be implemented to manage it depending on the type of information being passed.

Code Example:

For this exmaple, I used a local file called users.json containing an array of objects that in this case are representing random users. After clicking the button a foreach loop go through each object and display the user's full name and email address.
